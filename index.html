Метод Зейделя для решения СЛАУ

Gauss-Seidel Method for Solving SLAE

Метод Зейделя (или метод Гаусса-Зейделя) — это итерационный метод для решения систем линейных алгебраических уравнений (СЛАУ) вида \( Ax = b \), где A — квадратная матрица, x — вектор неизвестных, b — вектор свободных членов.

The Gauss-Seidel method is an iterative technique for solving systems of linear algebraic equations (SLAE) of the form \( Ax = b \), where A is a square matrix, x is the vector of unknowns, and b is the vector of constants.

Формула

Formula

На каждой итерации \( k+1 \), значение \( x_i^{(k+1)} \) вычисляется как:

On each iteration \( k+1 \), the value \( x_i^{(k+1)} \) is computed as:

\[ x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j=1}^{i-1} a_{ij} x_j^{(k+1)} - \sum_{j=i+1}^{n} a_{ij} x_j^{(k)} \right) \]

Условие сходимости

Convergence Condition

Метод сходится, если матрица A имеет диагональное преобладание: для каждой строки \( |a_{ii}| \geq \sum_{j \neq i} |a_{ij}| \), с строгим неравенством хотя бы в одной строке.

The method converges if matrix A is diagonally dominant: for each row \( |a_{ii}| \geq \sum_{j \neq i} |a_{ij}| \), with strict inequality in at least one row.

Пример

Example

Рассмотрим систему:

Consider the system:

\[ 4x_1 + x_2 = 5 \] \[ x_1 + 3x_2 = 4 \]

Начальное приближение: \( x^{(0)} = (0, 0) \), \(\epsilon = 0.01\).

Initial approximation: \( x^{(0)} = (0, 0) \), \(\epsilon = 0.01\).
